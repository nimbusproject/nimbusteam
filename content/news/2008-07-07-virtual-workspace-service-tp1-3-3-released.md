<div class="news-date">Jul 07, 2008</div>
<p>The main new feature provided in this release is the ability to deploy &#8220;one-click&#8221; virtual clusters &#8212; a much awaited release of the contextualization functions allowing users to create self-configurable virtual clusters on the fly. The new feature comes with improvements to the ensemble service and image compression facilities that extend the range of deployment scenarios in which it can be used.</p>
<p>See the <a href="https://workspace.globus.org/vm/TP1.3.3/index.html#changelog">changelog</a> for all the details.</p>
